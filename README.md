
# LexiFlow - LDOCE Inspired Dictionary

An elegant, high-performance English-to-English dictionary application powered by the **Google Gemini 1.5 Pro API**.

## ‚ú® Features

- **Longman Style**: Definitions written in simple English (Longman 2000-word vocabulary style).
- **Frequency Tags**: Identifies core English vocabulary (S1, W1, etc.).
- **Smart Suggestions**: Autocomplete as you type.
- **Bi-directional Search**: Query in English or Chinese.
- **Privacy-First Translation**: Chinese translations are hidden at the bottom of the page and only shown when requested.
- **Deep Navigation**: Click any word in a definition to search for it immediately.
- **Smart Backstack**: One-click navigation back to your previous search.

4. Run `npm run build` to build the project.
